# Architecture Overview

## Planned Stack
- Frontend: Next.js + JavaScript
- Styling: Tailwind CSS
- Backend/Database: Supabase
- Deployment: Vercel
- AI Summary Generation: OpenAI API

## Planned Core Flow

User Input Form
↓
Audit Engine Logic
↓
Savings Calculation
↓
AI Generated Summary
↓
Results Page
↓
Lead Capture + Public Shareable URL

## Notes
This architecture is intentionally focused on rapid MVP development, maintainability, and deployment speed within the assignment timeline.
